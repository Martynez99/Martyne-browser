# Martyne Browser

**MarTyne Browser - Pentesting + Privacy + AI tools**

A specialized web browser with integrated security testing, privacy protection, and AI-powered features for security professionals.

## Features

### 🔍 Recon & Info
- **Detect Server** - Identify web server technology
- **View Cookies** - Inspect cookie data
- **Extract Links** - Parse all links from a page
- **Fetch Headers** - Retrieve HTTP response headers
- **Execute JS** - Run custom JavaScript code

### ⚔️ Advanced Attacks
- **CGI Scanner** - Scan for CGI scripts
- **HTTP Brute Force** - Attempt credential enumeration
- **Directory Brute Force** - Discover hidden directories

### 📱 Social Media OSINT
- Quick access to major platforms (Twitter, Facebook, Instagram, LinkedIn, TikTok, Reddit, YouTube, Telegram, WhatsApp)
- Username-based OSINT search

### 🧩 Lua / RudaScript
- Custom script execution
- DOM injection capabilities
- Widget injection

### 🌐 Privacy Features
- **Web Tracking Protection** - Block trackers and fingerprinting
- **App Tracking Protection** - Control app-based tracking
- **Email Protection** - Phishing detection and tracking pixel blocking

### 🤖 AI Features
- Page summarization
- Vulnerability scanning
- Google dork suggestions
- HTTP header analysis

### 🛠️ Browser Tools
- Bookmarks management
- History tracking
- Extensions support
- Search engine selection (Google, Bing, DuckDuckGo)
- Zoom controls
- Developer tools

## Usage

1. Open `index.html` in a web browser
2. Enter a target URL in the address bar
3. Use the left panel tools for reconnaissance and testing
4. Switch between Live View, Page Source, and Logs tabs

## Installation

Simply clone and open in your browser:
```bash
git clone git@github.com:Martynez99/Martyne-browser.git
cd Martyne-browser
# Open index.html in your browser
```

## Browser Compatibility

Works on modern browsers (Chrome, Firefox, Edge, Safari) with JavaScript enabled.
